# FUTURE_DS_03
Interactive Power BI dashboard for marketing funnel analysis using e-commerce customer behavior data. Analyzed conversion stages, cart abandonment, revenue trends, brand performance, and purchase behavior to identify funnel drop-offs and provide actionable business recommendations.

# Marketing Funnel Analysis Dashboard

## Project Overview

This project focuses on analyzing e-commerce marketing funnel performance using Power BI. The dashboard tracks customer journey stages from product views to cart additions and final purchases to identify conversion drop-offs, customer behavior patterns, and opportunities to improve lead-to-customer conversion.

The analysis helps businesses understand:

* Funnel conversion performance
* Customer engagement behavior
* Cart abandonment trends
* Category and brand performance
* Revenue contribution
* Purchase trends over time

---

# Objective

The main objective of this project is to:

* Analyze customer conversion funnel stages
* Identify major drop-off points in the funnel
* Evaluate product category and brand performance
* Monitor revenue and purchase trends
* Provide actionable business recommendations to improve conversions

# Dashboard Features

## KPI Cards

Displays high-level business metrics for quick performance evaluation.

## Funnel Analysis

Visualizes customer journey:
View → Cart → Purchase

Helps identify conversion drop-offs.

## Category Performance Analysis

Shows top-performing product categories based on purchases.

## Brand Revenue Analysis

Analyzes revenue contribution from different brands.

## Purchase Trend Analysis

Tracks purchase behavior over time.

## Interactive Slicers

Allows filtering by:

* Brand
* Category
* Event Type
* Date Range

# Key Insights

* Significant drop-off exists between product views and cart additions.
* A small number of brands contribute to majority of purchase revenue.
* Certain product categories dominate customer purchases.
* Conversion rate remains low compared to overall traffic.
* Cart abandonment presents optimization opportunities.

---

# Recommendations

* Improve product page quality and descriptions.
* Simplify checkout experience.
* Retarget abandoned cart users.
* Increase promotions for high-converting brands.
* Focus marketing campaigns on top-performing categories.
